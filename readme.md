## Så här funkar det

- Skriv klart funktionerna i crudMember.ts tills alla testerna går igenom.
- Testerna finns i crudMember.spec.ts
- Kolla på testerna för att förstå vad funktionerna ska göra

## Installera

`npm install`

## Köra testerna

`npm test`

## Köra testerna med watch

`npm run test:watch`
